# 🗺 Sitemap Analyzer

Мощный инструмент рекурсивного обхода сайтмапов с визуальным деревом разделов.

## Возможности

- **Рекурсивный обход** — автоматически находит вложенные сайтмапы (index → urlset → urls)
- **Умное дерево** — группирует URL по сегментам пути, листовые узлы с ≥20 URL схлопываются в счётчик
- **Автораскрытие до 5 уровней** при старте, остальное collapsed
- **Поиск** по всему дереву с подсветкой
- **Гибкая параллельность** запросов (1–30)
- **Два вида**: дерево + таблица сайтмапов
- **Экспорт**: Markdown (для ИИ) + JSON (машиночитаемый)
- **MD3 Dark Theme** дизайн

## Установка и запуск

### Требования
- Node.js ≥ 18
- npm ≥ 9

### Шаги

```bash
# 1. Установить зависимости
npm install

# 2. Запустить
npm start

# Или в dev-режиме (с DevTools)
npm run dev
```

## Использование

1. Вставьте URL сайтмапа (поддерживаются все форматы):
   - `https://example.com/sitemap.xml` — простой URLset
   - `https://example.com/sitemap_index.xml` — индексный сайтмап
   - Любая глубина вложенности

2. Выберите параллельность запросов (ползунок)

3. Нажмите **▶ Анализировать**

4. Изучайте дерево:
   - Клик на строку — раскрыть/свернуть
   - Счётчик URLs у каждого узла
   - Листовые группы с 20+ URL — показ по клику на пилюлю
   - Поиск по URL в правом верхнем поле

5. Экспорт кнопками в боковой панели

## Форматы экспорта

### Markdown
- Заголовок с метаданными (дата, root URL, статистика)
- Таблица всех сайтмапов с типом и статусом
- Дерево разделов в текстовом виде
- Оптимизирован для подачи в LLM как контекст

### JSON
- Полная структура данных
- `tree` — иерархия разделов
- `allUrls` — плоский массив всех URL
- `sitemapsFlat` — список всех обойдённых сайтмапов
