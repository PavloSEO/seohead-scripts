# SEO Site Kit — Web UI Kit

Interactive demo of the MD3 web component library. Four clickable screens:

| Screen | Description |
|---|---|
| **Статья** (Article) | h1 + lead + TOC sidebar + prose + blockquotes + callouts + FAQ accordion + share block + related articles |
| **Главная** (Home) | Hero (centered variant) + stats strip + features grid + testimonials carousel |
| **Блог** (Blog) | Filter chips + article card grid |
| **Контакты** (Contact) | Contact form (RHF-ready) + office info |

## Components demonstrated

- Sticky header with scroll-shrink (80px → 64px)
- Breadcrumbs
- MD3 Chips (filter, assist)
- MD3 Cards (outlined, elevated)
- MD3 Buttons (filled, outlined, tonal)
- Accordion / FAQ
- Snackbar
- Cookie consent banner (GDPR)
- Author block
- Share block
- Article TOC sidebar (sticky)
- Star rating
- Contact form with validation hook
- Footer (4-column)

## Usage

Open `index.html` in a browser or serve via any static server.
The page selector at the top lets you switch between all demo screens.
Scroll position is preserved across page reloads via localStorage.
